# Идея телеграмм-бота для парковки

## Концепция

Телеграмм-бот для управления парковочными местами. Бот присутствует в чате парковки и помогает разыгрывать свободные места между участниками.

## Механизм работы

1. **Объявление свободного места**: Кто-то в чате пишет сообщение вида «место N сегодня свободно».

2. **Реакция бота**: Бот видит это сообщение и сразу же подключается, подтверждая, что место свободно.

3. **Кнопка участия**: Под сообщением бота появляется кнопка «я хочу!», позволяющая участникам чата выразить желание занять место.

4. **Сбор заявок**: Бот получает ID пользователя каждого, кто нажал на кнопку.

5. **Розыгрыш**: После прохождения определенного интервала времени (например, 2 минуты), бот случайным образом выбирает ID одного из пользователей.

6. **Уведомление победителя**: Бот пишет сообщение в чате с упоминанием выбранного пользователя и сообщает, что место теперь за ним.
